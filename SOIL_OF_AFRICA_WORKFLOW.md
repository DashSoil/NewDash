# Soil of Africa - Skills Development Centre Workflow Guide

## Overview
Soil of Africa is a Skills Development Centre that receives hundreds to thousands of CVs/applications. They need to:
1. Process CVs and create learner profiles
2. Create programs/learnerships (e.g., Beauty Therapy, Marketing)
3. Promote programs to attract learners
4. Connect with sponsors for funding
5. Enroll learners in specific courses
6. Communicate with learners throughout their journey

## Step-by-Step Workflow

### 1. **Bulk Import CVs/Applications**
**Location:** Hamburger Menu → Data Import → Bulk CV Import

**What to do:**
- Export CV data from your system (Excel/CSV)
- Ensure columns: `email`, `first_name`, `last_name`, `phone`, `id_number` (optional), `notes` (optional)
- Upload the CSV file
- System creates learner profiles automatically
- Handles duplicates (updates existing profiles)

**Result:** All applicants now have profiles in the system

---

### 2. **Create Programs/Learnerships**
**Location:** Programs Tab → "Create" Button (top-right)

**What to do:**
- Enter program title (e.g., "Beauty Therapy Learnership")
- System auto-generates course code (e.g., `ORG-BEAUTY`)
- Add description, duration, dates
- Set max students (if limited)
- **For sponsored programs:**
  - Toggle "Sponsored Program"
  - Enter sponsor name and contact
  - System tracks sponsor relationship

**Result:** Program is created and ready for promotion

---

### 3. **Promote Programs & Get Learners**
**Location:** Programs Tab → Tap any program → "Share & Get Code"

**Features:**
- **Program Code:** Learners can enter this code to register
- **QR Code:** Scan to register instantly
  - QR code includes program name, code, and organization
  - Works on social media, posters, flyers
- **Registration Link:** Share via WhatsApp, email, SMS
- **Social Media Templates:**
  - Facebook/Instagram post template (ready to copy/paste)
  - WhatsApp/SMS template with program code

**Marketing Channels:**
1. **Social Media:** Post QR code + program details
2. **WhatsApp Groups:** Send registration link
3. **Physical Posters:** Print QR code + program code
4. **Email Campaigns:** Include registration link
5. **Job Fairs/Events:** Display QR code for instant registration

**Result:** Learners can self-register using the code or QR code

---

### 4. **Bulk Enroll Existing Applicants**
**Location:** Enrollments Tab

**Two Options:**

#### Option A: Manual Enrollment (For CV Database)
- Tap "Enroll Manually"
- Select program
- Enter student details (or search existing)
- System auto-enrolls

#### Option B: Send Email Invites (For Active Recruitment)
- Tap "Send Invites"
- Select program
- Enter email addresses (bulk paste)
- System sends enrollment invitations
- Learners register themselves via link

**Use Case for Soil of Africa:**
- Import CVs → Manual Enrollment → Bulk enroll in "Beauty Therapy"
- Marketing campaign → Send invites → Learners self-register for "Marketing Course"

---

### 5. **Connect with Sponsors**
**Location:** Programs Tab → Create/Edit Program → "Sponsor & Additional Info"

**What to do:**
- When creating a sponsored program, add sponsor details
- System tracks sponsor relationship
- Can export reports showing sponsor-funded enrollments

**Future Enhancement:** Sponsor dashboard to show impact/ROI

---

### 6. **Communicate with Learners**
**Location:** Various screens

**Communication Methods:**
- **In-App Messaging:** Direct messages to learners
- **Announcements:** Broadcast messages to enrolled students
- **Notifications:** Push notifications for course updates

---

## Complete Workflow Example

### Scenario: Launching "Beauty Therapy Learnership" (Sponsored by Company X)

1. **Import Existing CVs:**
   - Go to Data Import → Bulk CV Import
   - Upload CSV with 500 CVs
   - System creates 500 learner profiles

2. **Create Program:**
   - Programs → Create
   - Title: "Beauty Therapy Learnership"
   - Code: `ORG-BEAUTY` (auto-generated)
   - Duration: 6 months
   - Enable "Sponsored Program"
   - Add sponsor: "Company X"

3. **Promote Program:**
   - Programs → Beauty Therapy → "Share & Get Code"
   - Download QR code
   - Get social media template
   - Post on Facebook/Instagram
   - Send WhatsApp messages with link
   - Print posters with QR code

4. **Enroll Learners:**
   - **From CV database:** Enrollments → Manual Enrollment → Select program → Enter details
   - **From marketing:** Learners scan QR or enter code → Auto-enroll via registration

5. **Track Progress:**
   - Dashboard shows enrolled students
   - Programs screen shows enrollment count
   - Can filter by sponsor

## Tips & Best Practices

1. **CSV Format:** Always use the template. Required: email, first_name, last_name
2. **Program Codes:** Keep them short and memorable (e.g., `ORG-BEAUTY`, `ORG-MARKET`)
3. **QR Codes:** Test scan before printing/posting
4. **Bulk Operations:** Process large imports during off-peak hours
5. **Communication:** Use in-app messaging for course updates
